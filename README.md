## Architecture

The project follows a scalable feature-based architecture:

src/
app/ # Next.js App Router structure
features/ # Domain-based modules (home, projects, contact)
components/ # Shared UI components
shared/ # Utilities, hooks, helpers
lib/ # Core utilities
types/ # TypeScript types

Each feature is self-contained and designed for scalability and maintainability in large applications.
